R&D CAPEX AUTOMATION TOOL

STEPS:

1. Place R&D Actuals file and WBS Template in this folder
2. Open config.xlsx and update:
   - Actuals file name
   - Sheet name
   - WBS file name
3. Save config.xlsx
4. Double-click rd_capex_automation_v1.exe
5. Output will be available in Output folder

NOTES:
- Do not rename columns inside input files
- Ensure sheet names are correct
